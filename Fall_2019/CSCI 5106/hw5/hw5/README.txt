In this directory includes:
 - The C file Problem 1 named hw51.c
 - The C file Problem 2 named hw52.c
 - The C file Problem 3 named hw53.c

Each file contains implementations, and tests for varying Binary Search Tree
implementations: hw51.c contains an implementation of an integer BST, hw52.c
contains an implementation of integer and string BST's using the union type, and
hw53.c contains and implementation of an arbitrary type BST using null pointers.
